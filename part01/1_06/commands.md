## Building main application

To rebuild the same image, first checkout the correct tag:

```
git checkout ex1.06
```

```
./docker_build_and_publish.sh ex1.06
```

## Create cluster

```
k3d cluster create --port 8082:30080@agent:0 -p 8081:80@loadbalancer --agents 2
```
